# Ecommerce-Sales-Dashboard-Excel
E-commerce sales analysis dashboard created using Excel

## Project Overview
This project is an Excel-based E-commerce Sales Dashboard created to analyze
sales performance, profitability, customer contribution, and regional trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Key Insights
- Sales peak in May and October, while December shows a decline.
- Clothing category contributes the highest sales and profit.
- South region records the highest sales compared to other regions.
- Top customers generate a significant portion of revenue.
